 从选料就上乘的健康欧式烘培店——多乐之日

 AUTHENTIC BAKERY

 TOUS LES JOURS


 “多乐之日”（Tour Les Jours）是韩国希杰集团旗下的烘焙品牌，以

 “健康美味，值得信赖，家庭式烘焙，经典欧式风格”著称。


   多乐之日是提供“家庭式烘焙”的面包蛋糕的健康的欧式烘焙店， 产品使用天然美味的原料制作。



 “Tous Les Jours”在法语中是“每天每日”的意思，我们每天都在店铺现场烤制美味的面包，使用高品质

   的面粉等原材料，保证面包更佳健康，美味；现场制作的蛋糕，能品尝到食材的天然美味。


   在多乐之日，有面包，蛋糕，曲奇，三明治，咖啡，冰淇淋等各种美味，明亮的灯光，舒适的桌椅，给人

   提供了舒服的感觉及放松交流的空间。


   目前多乐之日开拓直营店北京21家店，天津5家店，上海6家店，苏州1家店，威海1家店，在中国，多乐之

   日获得了“中国烘培优秀品牌”，“质量信誉服务AAA”及“明星饼店”等荣誉称号。
